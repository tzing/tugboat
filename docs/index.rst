Tugboat
=======

A linter to streamline your `Argo Workflows`_ with precision and confidence.

.. _Argo Workflows: https://argoproj.github.io/workflows/

.. attention::

   Tugboat is currently in development. Stay tuned for updates!


Installation
------------

Tugboat is currently only available on GitHub.
To install Tugboat, run the following command to install the latest version from the ``main`` branch:

.. code-block:: bash

   pip install git+https://github.com/tzing/tugboat.git#egg=tugboat
